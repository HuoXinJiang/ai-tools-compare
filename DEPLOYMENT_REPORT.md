### 📦 快速部署选项

#### 选项 1：GitHub Pages (推荐)

```bash
# 1. 安装 GitHub CLI (如果还没有)
brew install gh

# 2. 登录 GitHub
gh auth login

# 3. 创建并推送仓库
cd AI-Tools-Compare
gh repo create ai-tools-compare --public --push --source . --default-branch main

# 4. 启用 GitHub Pages
gh repo deploy
```

#### 选项 2：手动部署

1. 在 GitHub 创建新仓库 "ai-tools-compare" (公开)
2. 运行以下命令:
```bash
cd AI-Tools-Compare
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/ai-tools-compare.git
git push -u origin main
```
3. 在仓库设置中启用 GitHub Pages

#### 选项 3：Netlify/Vercel (一键部署)

```bash
# 安装 Netlify CLI
brew install netlify-cli

# 部署 (会自动处理)
netlify deploy --prod --dir=./AI-Tools-Compare --force
```
